# 掌机·阿西西 使用帮助

欢迎使用 `掌机·阿西西`。

这是一款运行在 HarmonyOS 上的复古掌机模拟器，目前主要支持：

- `GB`
- `GBC`
- `GBA`

如果你是第一次使用，建议先看完“快速开始”和“设置说明”两部分，再去导入游戏。

## 快速开始

### 1. 导入游戏

1. 打开首页。
2. 点击右上角的 `+` 按钮。
3. 在系统文件选择器里选择本地 ROM。
4. 导入完成后，游戏会自动出现在首页游戏库中。

说明：

- 当前一次最多可导入 `10` 个 ROM。
- 如果你打开了“新增 ROM 时自动同步封面”，导入后会顺便尝试匹配在线封面，所以会比单纯导入稍慢一点。

### 2. 开始游玩

1. 在首页点击游戏卡片。
2. 打开游戏操作弹窗后，点击 `开始游玩`。

### 3. 退出游戏

进入游戏后，可以通过下面两种方式打开游戏菜单：

- 点击虚拟手柄上的 `MENU`
- 使用系统返回键

然后在游戏菜单里点击 `退出`，就可以回到首页。

## 首页怎么用

首页是你的主游戏库，常用功能都集中在这里。

### 顶部按钮

右上角依次可以看到这些入口：

- `搜索`
- `添加游戏`
- `设置`
- `关于`

### 游戏分类

首页上方支持按类型切换：

- `全部`
- `收藏`
- `GB`
- `GBC`
- `GBA`

如果你已经导入了很多游戏，先用分类筛选会更方便。

### 排序方式

游戏库右侧支持切换排序方式，目前包括：

- `导入新到旧`
- `导入旧到新`
- `游玩时长`
- `最近打开`

简单理解：

- 想看刚导入的游戏，就选 `导入新到旧`
- 想继续最近玩的游戏，就选 `最近打开`
- 想把主力常玩游戏排前面，就选 `游玩时长`

### 游戏卡片怎么操作

点击游戏卡片后，会打开游戏操作弹窗。

你可以在这里进行：

- `开始游玩`
- `收藏 / 取消收藏`
- `编辑名称`
- `替换封面`
- `删除游戏`

补充说明：

- `编辑名称` 只会修改在本软件里的显示名称，不会改动原始 ROM 文件名
- `删除游戏` 会同时删除这个游戏对应的 ROM、存档、即时存档和当前封面，请谨慎使用

## 搜索怎么用

点击首页右上角 `搜索` 进入搜索页。

你可以直接输入下面这些内容来查找游戏：

- 中文名
- 英文名
- 文件名
- 机种名，例如 `GB`、`GBC`、`GBA`

搜索结果会实时刷新，不需要手动确认。

如果搜不到，建议换一种写法试试，例如：

- 中文名搜不到时改用英文名
- 游戏名搜不到时直接搜文件名

## 封面功能怎么用

游戏封面支持两种方式更新。

### 1. 在线更新封面

路径：

`首页 -> 点击游戏卡片 -> 封面 -> 在线更新封面`

适合场景：

- 导入后没有自动匹配成功
- 想换成更标准的盒封或标题封面

### 2. 手动选择图片

路径：

`首页 -> 点击游戏卡片 -> 封面 -> 手动选择图片`

说明：

- 手动选择的图片会自动按封面区域居中裁切
- 比较适合使用你自己收藏的海报图、截图或盒封图

## 游戏界面说明

进入游戏后，画面区域主要由这几部分组成：

- 游戏画面
- 虚拟手柄或 Delta 皮肤
- 游戏菜单
- 可选的 FPS 显示

### 默认虚拟手柄

在没有使用 Delta 皮肤时，软件会显示默认虚拟手柄。

特点：

- 会根据横屏或竖屏自动切换布局
- `GB / GBC` 这类不需要肩键的游戏，会尽量简化手柄区域
- 可以在游戏菜单里临时隐藏或重新显示

### Delta 皮肤

如果你启用了 Delta 皮肤，游戏里会优先显示你选中的皮肤布局。

如果当前情况没有可用布局，例如：

- 当前机种没有对应皮肤
- 当前方向没有对应布局

软件会自动退回默认虚拟手柄，不会影响继续游玩。

## 游戏菜单都有什么功能

在游戏中点击 `MENU` 或按系统返回键后，会打开游戏菜单。

常用功能如下。

### 继续

关闭菜单，回到当前游戏。

### 重启

重新启动当前游戏。

说明：

- 这是软重启
- 不会删除你的存档

### 退出

退出当前游戏并回到首页游戏库。

### 存档

把当前进度保存到即时存档槽位。

### 读档

从即时存档槽位恢复进度。

说明：

- 当前默认提供 `10` 个即时存档槽位
- 没用的槽位也可以在槽位面板里删除

### 金手指

打开金手指输入面板。

使用方式：

- 一行输入一条代码
- 保存后，下次可以继续使用

如果设置里启用了“自动应用已保存金手指”，下次进入同一个游戏时会自动尝试应用。

### 声音

快速静音或恢复游戏声音。

### 快进

快速切换快进开关。

关闭时是普通速度，开启后会按照你在设置里选好的倍率运行。

### 游戏震动

开启或关闭游戏本身的震动反馈。

### 音频震动

根据游戏声音节奏生成震动反馈。

适合那些本身没有原生震动效果的游戏。

### 滤镜

切换画面显示风格。

当前常见滤镜有：

- `像素`
- `平滑`
- `扫描线`

### 按键震动

控制虚拟手柄按下时的触感反馈。

### FPS

显示或隐藏右上角帧率。

### 虚拟手柄

显示或隐藏默认虚拟手柄。

说明：

- 使用 Delta 皮肤时，这个选项可能不会单独显示

### 手柄驱动

开启或关闭 USB 手柄驱动。

如果你接了外部 USB 手柄，通常建议打开。

## 设置说明

设置页是这款软件最重要的管理页面之一。大部分选项改完后会自动保存。

## 一、基础设置

### 默认开启音频

打开后，每次新进入游戏都会默认带声音。

### 默认开启快进

打开后，每次进入游戏都会自动开启快进。

如果你平时喜欢加速刷流程，这个选项会很方便。

### 快进倍率

可选：

- `2x`
- `3x`
- `4x`

建议：

- 想稳定一点，选 `2x`
- 想更快一些，选 `3x`
- `4x` 更适合测试或快速跳过重复流程

### 画面滤镜

可选：

- `像素`：画面更锐利，更接近原始掌机观感
- `平滑`：边缘更柔和，适合大屏显示
- `扫描线`：更有复古显示器味道

### 显示 FPS

打开后，游戏右上角会显示实时帧率。

## 二、输入与手柄

### Delta 皮肤管理

进入后可以：

- 开启或关闭 Delta 皮肤
- 导入新的 `.deltaskin`
- 为 `GBA` 选择默认皮肤
- 为 `GB / GBC` 选择默认皮肤
- 删除不需要的皮肤

关闭 Delta 皮肤后，游戏会回到默认虚拟手柄。

### USB 手柄驱动

这个选项用于启用外接 USB 手柄支持。

适合：

- Xbox 手柄
- DS4 / DS5
- Switch Pro
- 其他兼容的 USB 手柄

### 界面触感反馈

控制点击虚拟手柄或皮肤热区时，设备是否给出短促震动反馈。

### 界面触感强度

可选：

- `轻`
- `中`
- `强`

### 默认虚拟手柄高度

用来调整默认虚拟手柄在屏幕下方占用的高度。

说明：

- 只影响默认虚拟手柄
- 不影响 Delta 皮肤

## 三、震动与反馈

### 游戏震动

控制游戏本身的震动反馈。

### 游戏震动输出

可选：

- `设备`
- `USB 手柄`

意思很简单：

- `设备`：让手机或平板自己震动
- `USB 手柄`：让外接手柄震动

### 游戏震动强度

可选：

- `轻`
- `中`
- `强`

### 音频震动

根据游戏音乐、音效或节奏来生成震动。

适合没有原生震动支持的游戏。

### 音频震动输出

同样可以选：

- `设备`
- `USB 手柄`

### 音频震动强度

控制音频震动的力度。

### 音频震动语音过滤

这个选项用来减少人声或部分高频内容带来的误触发。

可选：

- `关闭`
- `低`
- `中`
- `高`

如果你觉得震动太频繁，建议试试 `中` 或 `高`。

## 四、存档与高级

### 新增 ROM 时自动同步封面

打开后，导入新游戏时会自动尝试联网匹配封面。

关闭后：

- 导入速度会更快
- 但封面需要你后面手动更新

### 自动应用已保存金手指

打开后，已经保存过的金手指会在下次进入同一个游戏时自动尝试应用。

### 记住上次使用的存档槽位

打开后，软件会记住你上次使用的即时存档槽位，方便下次继续。

### 显示存档缩略图

打开后，在存档列表里会显示预览图。

### 重置

设置页右上角有 `重置` 按钮。

如果你把设置改乱了，可以一键恢复为默认配置。

## Delta 皮肤怎么用

### 打开路径

`首页 -> 设置 -> Delta 皮肤管理`

### 基本使用步骤

1. 打开 `启用 Delta 皮肤`
2. 导入 `.deltaskin` 文件
3. 点击某个皮肤
4. 选择把它设为：
   - `GBA`
   - 或 `GB/GBC`

### 删除皮肤

如果删除了当前正在使用的皮肤，软件会：

- 自动切换到其他兼容皮肤
- 如果没有兼容皮肤，就回退到默认虚拟手柄

## 外接手柄怎么用

如果你打算用手柄玩，推荐优先打开 `USB 手柄驱动`。

开启路径：

- `设置 -> USB 手柄驱动`
- 或 `游戏菜单 -> 手柄驱动`

如果手柄没有反应，可以检查：

1. `USB 手柄驱动` 是否已经打开
2. 手柄是否已经正确连接
3. 当前是否在游戏界面中
4. 是否被系统或其他输入方式占用

## 常见问题

### 为什么导入游戏有时会慢一点？

常见原因有：

- 一次导入了多个 ROM
- 开启了自动同步封面
- 首次匹配在线封面时需要初始化数据

如果你更在意导入速度，建议关闭：

- `新增 ROM 时自动同步封面`

### 为什么在线更新封面后，列表没有马上变化？

一般是因为列表刷新稍有延迟。

可以尝试：

- 重新打开当前游戏卡片
- 切换一次分类
- 稍等几秒钟

### 为什么有些游戏没有震动？

因为并不是所有游戏都自带原生震动。

这时你可以尝试：

- 打开 `游戏震动`
- 或改用 `音频震动`

### 为什么 Delta 皮肤没有显示？

请确认：

1. 已开启 `启用 Delta 皮肤`
2. 当前机种已经选择了默认皮肤
3. 当前方向有对应布局

如果这些条件不满足，软件会自动显示默认虚拟手柄。

### 即时存档和游戏存档有什么区别？

- `即时存档`：随时可以存，随时可以读，更方便
- `游戏存档`：游戏本身自带的正常保存方式

## 推荐使用顺序

如果你第一次用，建议这样开始：

1. 先去设置里确认音频、快进和滤镜
2. 导入游戏 ROM
3. 视需要更新封面
4. 进入游戏体验默认虚拟手柄
5. 如果想换风格，再导入 Delta 皮肤
6. 如果要接外部手柄，再开启 USB 手柄驱动

这样最不容易踩坑，也最容易快速上手。
