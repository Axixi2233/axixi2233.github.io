# 遥光·阿西西 使用说明书

## 1. 项目简介

遥光·阿西西是一款基于 HarmonyOS 的 PlayStation 串流客户端，支持局域网串流、远程直连串流、PSN 远程串流，并提供虚拟手柄、USB 手柄接管、体感输入、触控板和性能信息叠加层等能力。

---

## 2. 功能总览

- 局域网主机发现与注册
- 主机卡片管理（账号切换、远程入口、唤醒）
- 本地串流 / 远程直连 / PSN 远程三种模式
- 串流页游戏菜单（退出并待机、触控板、布局编辑、画面位置）
- 虚拟手柄（支持布局编辑、横竖屏分别保存）
- USB 手柄接管（DS4/DS5/Xbox/NS Pro/NS2 Pro/Joy-Con 2）
- 设备体感与 USB 手柄体感切换
- 多层性能信息（简版 + 完整）
- 渲染与超分（系统渲染 / GLES XEngine，2K/4K）

---

## 3. 环境与安装

### 3.1 运行环境

- HarmonyOS 设备（手机、平板、2in1）
- 已开启远程游玩能力的 PS 主机
- 可访问同一局域网或公网（远程场景）

### 3.2 权限建议

- 网络权限（发现、连接、远程）
- USB 设备访问权限（USB 手柄接管）
- 传感器权限（设备体感）
- 麦克风权限（启用串流麦克风时）

### 3.3 网络建议

- 局域网模式优先使用 5G/千兆网络
- 远程模式优先保证上行稳定
- IPv6 域名远程直连已支持双栈解析；远程唤醒同样支持双栈尝试

---

## 4. 首次使用快速开始

1. 打开首页，点击右上角添加主机。
2. 在添加页填写目标主机信息并注册账号。
3. 返回首页，确认主机卡片出现且状态正常。
4. 点击开始串流进入本地串流，或点击远程串流进入远程流程。
5. 首次进入串流页可在设置页微调分辨率、码率、编码和输入选项。

---

## 5. 首页与主机卡片说明

### 5.1 卡片显示逻辑

- 同一台主机只显示一张卡片。
- 卡片显示主机地址、账号名称、PSN ID（脱敏）、主机 ID、更新时间等信息。

### 5.2 卡片操作按钮

- 远程串流
- 开始串流
- 本地唤醒
- 删除主机
- 切换账号（多账号时）

### 5.3 状态说明

- 待机中，可唤醒（可远程唤醒或本地唤醒）
- 在线（可直接开始串流）
- 离线或不可达（请检查网络、主机电源和账号信息）

## 6. 主机添加与账号管理

### 6.1 添加主机

- 添加页中的“账号名称”为必填字段，用于卡片展示。
- PSN account ID 可默认带入已绑定账号，支持后续切换。

### 6.2 本地账号管理

- 设置页进入“本地账号管理”。
- 支持按主机查看本地账号列表。
- 支持删除账号（带确认弹窗）。

### 6.3 账号切换

- 首页主机卡片支持切换账号。
- 切换后会影响下次串流使用的账号。

### 6.4 PSN Account ID（Base64）获取

- 推荐方式：在 `PSN 账号管理` 中完成登录或导入重定向 URL，应用会自动拿到 `user_id` 并计算 `Account ID (Base64)`，无需手动转换。
- 手动方式：可通过 [https://psn.flipscreen.games/](https://psn.flipscreen.games/)或者  [https://www.psntools.com/psn/checker/](https://www.psntools.com/psn/checker/) 查询 `PSN Account ID (Base64)`。
- 特别注意：添加主机页填写 **Base64 格式** 的 Account ID，不要填写十进制 `user_id`。
- (注)添加主机页已支持自动转换：如果输入的是纯数字 `user_id`，提交注册时会自动转成 Base64 并回填。
- 示例：`4201313478908013781` 会自动转换为 `1Y/JW1d6sUs=`。

## 7. 远程串流模式说明

### 7.1 本地串流

- 适用于同局域网，延迟最低、稳定性最好。

### 7.2 直连远程

- 输入远程 IP 或域名。
- 支持远程唤醒与远程连接分支。
- 支持保存远程地址并回显。

直连远程建议配置路由器端口映射（公网 -> 主机内网 IP）：

- `TCP 9295`：会话初始化（Session Request）
- `UDP 9296`：媒体流（视频/音频/输入）
- `UDP 9297`：可选兼容端口（部分网络/固件环境建议同时放行）
- `UDP 9302`：PS5 唤醒/发现相关端口
- `UDP 987`：PS4 唤醒/发现相关端口

说明：

- 如果仅使用 PS5，可优先映射 `9295/TCP + 9296/UDP + 9302/UDP`。
- 如果仅使用 PS4，可优先映射 `9295/TCP + 9296/UDP + 987/UDP`。
- `9303-9319` 是本地发现套接字使用的本地端口范围，通常不需要手动映射为公网转发端口。
- 若公网为运营商 CGNAT，端口映射可能失效，建议改用 IPv6、内网穿透或 PSN 远程模式。

### 7.3 PSN 远程

- 通过 PSN 账号完成远程设备查询和建链。
- 与直连远程互不冲突，可按场景选择。

## 8. PSN 远程配置与登录

### 8.1 入口

- 设置页 `账号与远程` -> `PSN 账号管理`

### 8.2 登录方式

- 登录 PSN（内置浏览器自动导入）
- 导入重定向 URL（手动导入）

手动导入时，请确保重定向 URL 形如：

- `https://remoteplay.dl.playstation.net/remoteplay/redirect?...&code=...`

应用会从 URL 中提取 `code`，再换取 `Access Token / Refresh Token / Online ID / Account ID (Base64)`。

### 8.3 列表与状态

- 显示 Online ID、账号 ID（脱敏）、Token 有效期、状态
- 支持刷新登录状态
- 支持删除账号（带确认弹窗）

## 9. 串流页面使用说明

### 9.1 返回行为

- 串流未成功建立时可直接返回。
- 串流成功后返回会打开游戏菜单。

### 9.2 游戏菜单

- 退出并待机
- 触控板开关
- 触控板键
- 编辑手柄
- 画面位置（居中/顶部居中/底部居中）
- 画面边距（0-300）

### 9.3 画面位置

- 仅在当前会话中生效（不持久化）。

### 10.1 基本能力

## 12. 超分与渲染设置

### 12.1 渲染方式

- 系统渲染：AVCodec 直出，稳定优先
- GLES XEngine：可叠加 GPU 超分

### 12.2 超分目标

- 关闭
- 2K
- 4K

### 12.3 生效条件

- 仅 720p / 1080p 输入支持超分
- 超分失败会自动回退到普通 GLES 渲染链

### 12.4 使用建议

- 低码率场景下超分观感通常更稳
- 高码率和复杂运动场景可能出现局部抖动，可降低码率或切回系统渲染验证

## 16. 常见问题排查

### 16.1 远程连接失败

- 检查账号是否可用、目标地址是否正确、主机是否可达。
- 直连远程建议先测试 IPv4，再测试域名与 IPv6。

### 16.2 远程唤醒失败

- 已支持双栈唤醒尝试，但公网链路对 UDP 更敏感。
- 请确认路由转发、主机待机网络设置和运营商网络策略。

### 16.3 PSN 远程失败

- 优先检查 Token 是否过期。
- 若日志出现 `Unknown Session Request Error`，通常卡在会话响应阶段，建议保留完整日志反馈。

### 16.4 画面卡顿/色块抖动

- 降低码率验证网络瓶颈。
- 在系统渲染与 GLES 渲染之间切换对比。
- 超分场景优先搭配较低码率观察稳定性。

### 16.5 手柄异常

- 确认已开启自动 USB 接管或手动启动成功。
- 核对 USB 手柄上报率与体感输入源设置。
- 需要时进入手柄测试页做按键/摇杆/扳机/体感验证。

## 18. 已知限制与兼容性说明

- 部分公网环境下远程唤醒成功率低于远程连接。
- 不同设备 GPU/驱动对 GLES 超分表现存在差异。
- 某些手柄高级能力（如特定触觉路径）受设备与系统接口限制。

---

## 19. 隐私与安全说明

- 账号授权信息与主机记录保存在本地。
- 手册建议仅导入可信来源的重定向 URL。
- 对外网络请求主要用于 PSN 登录与远程建链能力。

## 20. 版本更新记录与术语表

### 20.1 更新记录（建议）

- 每次发布时“新增、优化、修复、已知问题”四部分。

### 20.2 常见术语

- 本地串流：设备与主机在同局域网内连接。
- 直连远程：通过公网地址（IP/域名）直接连接主机。
- PSN 远程：通过 PSN 账号链路完成远程连接流程。
- 超分：对输入画面做空间放大与锐化处理。
- 体感：陀螺仪/加速度计等传感器输入。
